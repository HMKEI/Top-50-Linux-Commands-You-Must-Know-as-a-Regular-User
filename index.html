Top 50 Linux Commands You Must Know as a Regular User

1. ls

Display a list of files and directories in the current directory.

يعرض قائمة الملفات والمجلدات في الدليل الحالي.

bash
ls
خيارات شائعة:
ls -l : عرض الملفات بتفاصيل إضافية (الحجم، التاريخ، الأذونات).
ls -a : عرض جميع الملفات، بما في ذلك الملفات المخفية.

2. pwd

Print the current working directory.

يعرض المسار الكامل للدليل الحالي.

bash
pwd

3. cd

Change the current directory.

يستخدم لتغيير الدليل الحالي.

bash
cd <directory_name>
أمثلة:
cd /home/user : الانتقال إلى دليل /home/user.
cd .. : الرجوع إلى الدليل السابق.

4. mkdir

Create new directories.

إنشاء مجلد جديد.

bash
mkdir <directory_name>
أمثلة:
mkdir projects : إنشاء مجلد باسم projects.

5. mv

Move or rename files and directories.

نقل أو إعادة تسمية الملفات والمجلدات.

bash
mv <source> <destination>
أمثلة:
mv file1.txt folder/ : نقل الملف file1.txt إلى المجلد folder.
mv old_name.txt new_name.txt : إعادة تسمية الملف.

6. rm

Remove files or directories.

حذف الملفات أو المجلدات.

bash
rm <file_name>
أمثلة:
rm file1.txt : حذف الملف file1.txt.

7. rmdir

Remove an empty directory.

حذف مجلد فارغ.

bash
rmdir <directory_name>
أمثلة:
rmdir folder : حذف المجلد folder (إذا كان فارغًا).

8. touch

Create a new empty file or update the timestamp of an existing file.

إنشاء ملف فارغ أو تحديث الطابع الزمني لملف موجود.

bash
touch <file_name>
أمثلة:
touch newfile.txt : إنشاء ملف جديد باسم newfile.txt.

9. cat

Concatenate and display file content.

دمج وعرض محتوى الملفات.

bash
cat <file_name>
أمثلة:
cat file1.txt : عرض محتوى الملف file1.txt.

10. cp

Copy files or directories.

نسخ الملفات أو المجلدات.

bash
cp <source> <destination>
أمثلة:
cp file1.txt /home/user/ : نسخ الملف file1.txt إلى المجلد /home/user/.

11. man

Display the manual for a command.

عرض دليل الاستخدام للأوامر.

bash
man <command>
أمثلة:
man ls : عرض دليل الأمر ls.

12. history

Display the history of commands used.

عرض تاريخ الأوامر التي تم استخدامها.

bash
history

13. clear

Clear the terminal screen.

مسح شاشة الطرفية.

bash
clear

14. grep

Search for a pattern in files.

البحث عن نمط معين في الملفات.

bash
grep <pattern> <file>
أمثلة:
grep 'text' file1.txt : البحث عن النص "text" في الملف file1.txt.

15. echo

Display a line of text or a variable's value.

عرض سطر من النص أو قيمة متغير.

bash
echo <text>
أمثلة:
echo 'Hello World' : عرض النص "Hello World".

16. find

Search for files in a directory hierarchy.

البحث عن الملفات ضمن هيكل المجلدات.

bash
find <directory> -name <filename>
أمثلة:
find /home/user -name '*.txt' : البحث عن جميع الملفات ذات الامتداد .txt في المجلد /home/user.

17. tar

Archive files into a tarball (compressed file).

أرشفة الملفات إلى ملف مضغوط.

bash
tar -cvf archive.tar <file1> <file2>
أمثلة:
tar -cvf archive.tar file1.txt file2.txt : إنشاء أرشيف tar يتضمن الملفين file1.txt و file2.txt.

18. unzip

Extract compressed zip files.

استخراج الملفات المضغوطة بتنسيق zip.

bash
unzip archive.zip

19. df

Display disk space usage.

عرض استخدام المساحة على القرص.

bash
df

20. du

Display disk usage for files and directories.

عرض استخدام المساحة في الملفات والمجلدات.

bash
du -sh <directory>

21. ps

Display information about running processes.

عرض معلومات عن العمليات الجارية.

bash
ps

22. kill

Terminate a process.

إنهاء عملية معينة.

bash
kill <PID>
أمثلة:
kill 1234 : إنهاء العملية التي رقم معرفها 1234.

23. chmod

Change file permissions.

تغيير أذونات الملفات.

bash
chmod <permissions> <file_name>
أمثلة:
chmod 755 file1.txt : تعيين الأذونات 755 للملف file1.txt.

24. chown

Change file owner and group.

تغيير مالك الملف والمجموعة.

bash
chown <user>:<group> <file_name>
أمثلة:
chown user:group file1.txt : تغيير مالك الملف إلى "user" والمجموعة إلى "group".

25. ln

Create hard and symbolic links.

إنشاء روابط صلبة ورمزية.

bash
ln <source> <link_name>
أمثلة:
ln -s /path/to/file symlink : إنشاء رابط رمزي.

26. wget

Download files from the internet.

تحميل الملفات من الإنترنت.

bash
wget <URL>
أمثلة:
wget http://example.com/file.zip : تحميل الملف من الرابط.

27. curl

Transfer data from or to a server.

نقل البيانات من أو إلى الخادم.

bash
curl <URL>
أمثلة:
curl http://example.com : جلب محتوى صفحة الويب.

28. alias

Create a shortcut for a command.

إنشاء اختصار لأمر معين.

bash
alias <alias_name>=<command>
أمثلة:
alias ll='ls -l' : إنشاء اختصار للأمر ls -l باسم ll.

29. unalias

Remove a shortcut (alias).

إزالة الاختصار (alias).

bash
unalias <alias_name>
أمثلة:
unalias ll : إزالة الاختصار ll.

30. uname

Display system information.

عرض معلومات النظام.

bash
uname -a
أمثلة:
uname -a : عرض معلومات النظام التفصيلية.

31. top

Display real-time system processes.

عرض العمليات الجارية في النظام بشكل مباشر.

bash
top

32. htop

Interactive process viewer (requires installation).

عرض العمليات الجارية بشكل تفاعلي (يتطلب التثبيت).

bash
htop

33. sudo

Execute a command as a superuser or another user.

تنفيذ أمر كمستخدم متميز (سوبر يوزر) أو مستخدم آخر.

bash
sudo <command>
أمثلة:
sudo apt-get update : تحديث حزم النظام باستخدام صلاحيات الجذر.

34. apt

Package management tool for Debian-based distributions.

أداة إدارة الحزم للتوزيعات المبنية على ديبيان.

bash
sudo apt install <package_name>
أمثلة:
sudo apt install git : تثبيت أداة Git.

35. yum

Package management tool for Red Hat-based distributions.

أداة إدارة الحزم للتوزيعات المبنية على ريد هات.

bash
sudo yum install <package_name>
أمثلة:
sudo yum install git : تثبيت أداة Git على أنظمة ريد هات.

36. nano

Text editor for the terminal.

محرر نصوص يعمل في الطرفية.

bash
nano <file_name>
أمثلة:
nano file1.txt : فتح الملف file1.txt باستخدام محرر nano.

37. vim

Advanced text editor for the terminal.

محرر نصوص متقدم يعمل في الطرفية.

bash
vim <file_name>
أمثلة:
vim file1.txt : فتح الملف file1.txt باستخدام محرر vim.

38. sed

Stream editor for modifying files or input.

محرر تدفق لتعديل الملفات أو الإدخال.

bash
sed 's/<old_text>/<new_text>/' <file_name>
أمثلة:
sed 's/old/new/' file.txt : استبدال النص "old" بـ "new" في الملف file.txt.

39. awk

Pattern scanning and processing language.

لغة مسح الأنماط ومعالجة البيانات.

bash
awk '{print $1}' <file_name>
أمثلة:
awk '{print $1}' file.txt : طباعة العمود الأول من الملف file.txt.

40. cut

Remove sections from each line of files.

إزالة أجزاء من كل سطر في الملفات.

bash
cut -d' ' -f1 <file_name>
أمثلة:
cut -d' ' -f1 file.txt : استخراج العمود الأول من الملف file.txt.

41. sort

Sort lines of text files.

ترتيب أسطر النصوص في الملفات.

bash
sort <file_name>
أمثلة:
sort file.txt : ترتيب أسطر الملف file.txt.

42. uniq

Report or omit repeated lines.

إبلاغ أو إزالة الأسطر المكررة.

bash
uniq <file_name>
أمثلة:
uniq file.txt : إزالة الأسطر المكررة من الملف file.txt.

43. wc

Print the word, line, character, or byte counts.

طباعة عدد الكلمات أو الأسطر أو الأحرف أو البايتات.

bash
wc <file_name>
أمثلة:
wc file.txt : عرض عدد الأسطر والكلمات والأحرف في الملف file.txt.

44. tr

Translate or delete characters.

ترجمة أو حذف الأحرف.

bash
tr 'a-z' 'A-Z' <file_name>
أمثلة:
tr 'a-z' 'A-Z' file.txt : تحويل الأحرف من صغيرة إلى كبيرة في الملف file.txt.

45. diff

Compare files line by line.

مقارنة الملفات سطرًا بسطر.

bash
diff <file1> <file2>
أمثلة:
diff file1.txt file2.txt : مقارنة الملفين file1.txt و file2.txt.

46. cmp

Compare two files byte by byte.

مقارنة ملفين بايت بايت.

bash
cmp <file1> <file2>
أمثلة:
cmp file1.txt file2.txt : مقارنة الملفين file1.txt و file2.txt بايت بايت.

47. find

Search for files in a directory hierarchy.

البحث عن ملفات في هيكل الدليل.

bash
find /path/to/dir -name '<pattern>'
أمثلة:
find /home/user -name '*.txt' : البحث عن جميع الملفات ذات الامتداد .txt في دليل /home/user.

48. locate

Find files by name.

العثور على الملفات باستخدام الاسم.

bash
locate <filename>
أمثلة:
locate file1.txt : العثور على جميع الأماكن التي يوجد فيها الملف file1.txt.

49. updatedb

Update the database used by locate.

تحديث قاعدة البيانات التي يستخدمها locate.

bash
sudo updatedb

50. shutdown

Shut down or restart the system.

إيقاف تشغيل النظام أو إعادة تشغيله.

bash
sudo shutdown -h now
أمثلة:
sudo shutdown -h now : إيقاف تشغيل النظام فورًا.
